# pengExpressMS

#### 介绍
鹏城速运项目

#### 软件架构

鹏城项目是一套为物流公司提供后台管理的自动化平台。主要是对公司人员、车辆、订单的统一管理。

1. 项目整体架构

>> 前端：Vue框架

>> 后端：SpringBoot

2. 数据库

>> 关系型数据库：Mysql

>> 非关系型数据库：Redis、MongoDB

3. 项目演示

http://47.103.219.13 项目演示地址

#### 安装教程

1.  Mysql 安装
2.  Redis 安装
3.  MongoDB 安装

#### 使用说明

1.  前端：安装Nodejs,配置好环境后，输入命令，npm install、 npm run dev
2.  后端：找到根目录下的 PengMsApplication 启动类 启动。
